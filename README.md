# 🎓 UFC AVA POC — Plataforma de Aprendizado

```bash
# 1. Clonar e instalar
git clone <seu-repo> && cd ava-poc-react && pnpm install

# 2. Rodar aplicação React
cd apps/poc-react-vite && pnpm dev

# 3. Rodar aplicação Vue
cd apps/poc-vue-vite && pnpm dev

# 4. Buildar aplicação
pnpm build

```
